- Demandware OCAPI is not publicly accessible without partner credentials.
- Future option: Use the Open Commerce API (OCAPI) if vendor provides a client_id.

- **Lorna Jane**: Next.js custom app. The `/collections/` Shopify path returns 404; their product listing is rendered client-side.
- **Cotton On**: Custom platform with heavy anti-clickjack and bot detection.
- **Anaconda**: Unknown platform, connection timed out during research.
